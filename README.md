# Xtream Codes Player

This project is a lightweight web-based player for Xtream Codes API, compatible with Amazon Silk browser.

## Features
- **Login Support**: Enter your Xtream Codes credentials to fetch content.
- **Channel Display**: View available channels.
- **EPG Parsing**: Display an Electronic Program Guide (EPG) for live events.

## Files
- `index.html`: Main HTML layout.
- `styles.css`: Basic styling.
- `app.js`: Fetch and display channel data.
- `epg.js`: Fetch and parse EPG XML data.

## How to Use
1. Deploy the project on a server or open `index.html` in Silk browser.
2. Enter your Xtream Codes API credentials (username, password, and server URL).
3. View the fetched channels and EPG.

## Setup
No additional setup is required for plain HTML/JavaScript. Simply upload the files to your web server.

## Notes
- Ensure your Xtream Codes server URL is accessible via HTTPS for secure connections.
- Currently designed for Silk browser and similar lightweight browsers.
